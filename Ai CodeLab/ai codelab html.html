HTML Basics for Beginners

Welcome to AI Code Lab's HTML basics guide! This comprehensive tutorial will help you learn HTML from scratch with simple, easy-to-follow examples.

Introduction to HTML

HTML stands for Hypertext Markup Language. It's the standard language used to create web pages and applications. HTML is not a programming language; it's a markup language that tells web browsers how to structure the content on your page.

What you'll learn: By the end of this guide, you'll be able to create your own web pages with properly structured content, images, links, tables, and forms.

Why Learn HTML?

HTML is the foundation of all web pages. Without HTML, you wouldn't be able to organize text or add images or videos to your web pages. HTML is the beginning of everything you need to know to create engaging web pages!

HTML Document Structure

Every HTML document follows a basic structure. Let's look at the simplest HTML page:

<!DOCTYPE html> <html> <head> <title>My First Web Page</title> </head> <body> <h1>Hello, World!</h1> <p>This is my first web page.</p> </body> </html>

Understanding the Structure:

  • <!DOCTYPE html> - Declares the document type and HTML version
  • <html> - The root element that wraps all content on the page
  • <head> - Contains meta-information about the document
  • <title> - Sets the title that appears in the browser tab
  • <body> - Contains all the content that users see on the page

Basic HTML Elements

HTML uses "elements" to structure content. Elements are defined by tags, which are keywords surrounded by angle brackets like <tagname>.

HTML Tags:

Most HTML elements have an opening tag and a closing tag:

<tagname>Content goes here...</tagname>

Self-closing Tags:

Some elements don't need a closing tag because they don't contain any content:

<br> <img src="image.jpg">
HTML5 syntax: In HTML5, you can write self-closing tags as <img src="image.jpg" /> but the trailing slash is optional.

Headings and Paragraphs

Headings and paragraphs are the most basic elements for organizing text content.

Headings:

HTML provides six levels of headings, from <h1> (the most important) to <h6> (the least important).

<h1>This is a Main Heading</h1> <h2>This is a Level 2 Heading</h2> <h3>This is a Level 3 Heading</h3> <h4>This is a Level 4 Heading</h4> <h5>This is a Level 5 Heading</h5> <h6>This is a Level 6 Heading</h6>

HTML provides several elements to format text and give it special meaning.

<p><strong>Bold text</strong> is used for important content.</p> <p><em>Italic text</em> is used for emphasized content.</p> <p><mark>Highlighted text</mark> draws attention to specific parts.</p> <p><small>Smaller text</small> is used for fine print.</p> <p>This text contains <sub>subscript</sub> and <sup>superscript</sup>.</p>

Links and Anchors

Links are created using the <a> tag:

<a href="https://www.example.com">Visit Example.com</a> <a href="#top">Back to top</a> <a href="mailto:example@email.com">Send email</a>

Adding Images

Images are added using the <img> tag:

<img src="image.jpg" alt="Description of image">

Lists

HTML supports ordered, unordered, and description lists:

<ul> <li>Unordered item 1</li> <li>Unordered item 2</li> </ul> <ol> <li>Ordered item 1</li> <li>Ordered item 2</li> </ol>

Tables

HTML tables are used to display data in rows and columns:

<table> <thead> <tr> <th>Header 1</th> <th>Header 2</th> </tr> </thead> <tbody> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </tbody> </table>

Basic Forms

HTML forms are used to collect user input:

<form action="/submit" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> <button type="submit">Submit</button> </form>

Divs and Spans

Container elements for grouping and styling content:

<div class="container"> <p>This is a block-level <span class="highlight">container</span></p> </div>

Semantic Elements

HTML5 semantic elements provide meaning to the structure:

<header>Site header</header> <nav>Navigation menu</nav> <main>Main content</main> <article>Article content</article> <aside>Sidebar content</aside> <footer>Site footer</footer>

Introduction to CSS

CSS (Cascading Style Sheets) is used to style HTML elements:

<style> .container { background-color: #f0f0f0; padding: 20px; border-radius: 5px; } </style>
