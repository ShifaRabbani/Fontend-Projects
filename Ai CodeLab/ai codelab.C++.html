Introduction to C++

C++ is a high-performance, compiled programming language known for its efficiency and flexibility. Created by Bjarne Stroustrup in 1983, C++ is widely used in system programming, game development, competitive programming, embedded systems, and more.

Why Learn C++?

  • High Performance – Faster execution compared to interpreted languages.
  • Versatile – Used for game development, operating systems, and real-time applications.
  • Object-Oriented – Supports OOP principles like encapsulation and inheritance.
  • Strong Community Support – Extensive resources, libraries, and frameworks.
  • Career Opportunities – In demand for software development and system programming.
  • Cross-Platform – Runs on Windows, macOS, and Linux.

Installing C++

  1. Download and install a C++ compiler such as MinGW (Windows) or use g++ (Linux/macOS).
  2. Verify installation by running:
    3. g++ --version
  3. Use an IDE like Code::Blocks, Visual Studio Code, or Dev-C++ for writing C++ programs.

Variables and Data Types in C++

In C++, a variable is a named storage location in memory. You must declare a variable with its data type before using it. For example, string name = "Alice"; stores the text "Alice" in the variable name. C++ has various data types:

Integer: Whole numbers e.g., int x = 10;

Float: Decimal numbers e.g., float price = 10.5;

Double: More precise decimal numbers e.g., double pi = 3.14159;

Character: Single character e.g., char grade = 'A';

Boolean: True or False values e.g., bool isRaining = false;

C++ is a statically typed language, meaning you must specify the data type when declaring a variable. You can use the typeid() function (from the typeinfo library) to check a variable’s type.

#include <iostream>
                using namespace std;
                
                int main() {
                    string name = "Shifa";
                    int age = 20;
                    bool isStudent = true;
                
                    cout << "Name: " << name << endl;
                    cout << "Age: " << age << endl;
                    cout << "Is Student: " << isStudent << endl;
                
                    return 0;
                }

Control Statements in C++

Control statements help us control the flow of a program based on conditions and loops. There are two main types:

  • If-Else Statements – Used to make decisions
  • Loops – Used to repeat a block of code

1️⃣ If-Else Statement

An if-else statement checks a condition and decides what to do based on whether the condition is true or false.

#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;

    if (age >= 18) {
        cout << "You are an adult." << endl;
    } else {
        cout << "You are a minor." << endl;
    }
    return 0;
}

How it works?

  • The program asks the user to enter their age.
  • The if condition checks if the age is 18 or more.
  • If true, it prints "You are an adult."
  • If false, it prints "You are a minor."

💡 Use case: If-Else is useful when making decisions, like checking if a user is old enough to vote.

Operators in C++

Operators in C++ are symbols that perform operations on variables and values. C++ supports different types of operators.

🔹 Types of Operators in C++

  • Arithmetic Operators
  • Relational (Comparison) Operators
  • Logical Operators
  • Bitwise Operators
  • Assignment Operators
  • Increment & Decrement Operators
  • Conditional (Ternary) Operator

📌 1. Arithmetic Operators

These operators perform mathematical operations.

Operator Description Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division a / b
% Modulus (Remainder) a % b

📌 2. Relational (Comparison) Operators

These operators compare two values.

Operator Description Example
== Equal to a == b
!= Not equal to a != b
> Greater than a > b
< Less than a < b
>= Greater than or equal to a >= b
<= Less than or equal to a <= b

📌 3. Logical Operators

These operators are used to combine conditional expressions.

Operator Description Example
&& Logical AND (a > 5) && (b < 10)
|| Logical OR (a > 5) || (b < 10)
! Logical NOT !(a == b)

📌 4. Bitwise Operators

These operators perform operations at the bit level.

Operator Description Example
& Bitwise AND a & b
| Bitwise OR a | b
^ Bitwise XOR a ^ b
~ Bitwise NOT ~a
<< Left Shift a << 1
>> Right Shift a >> 1

📌 5. Assignment Operators

These operators assign values to variables.

Operator Description Example
= Assign a = b
+= Add and assign a += b
-= Subtract and assign a -= b
*= Multiply and assign a *= b
/= Divide and assign a /= b
%= Modulus and assign a %= b

📌 6. Increment & Decrement Operators

These operators increase or decrease a variable’s value.

  • ++ (Increment) → Increases value by 1
  • -- (Decrement) → Decreases value by 1

📌 7. Conditional (Ternary) Operator

A shorthand way of writing an if-else statement.

result = (a > b) ? a : b;

✅ Summary

Operators in C++ are essential for performing calculations, comparisons, and logical operations. Understanding them is key to writing efficient C++ programs.

Constructors and Destructors in C++

In C++, constructors and destructors are special member functions used for initializing and cleaning up objects automatically.

🔹 What is a Constructor?

  • A constructor is a special function that is automatically called when an object is created.
  • It has the same name as the class and does not have a return type.

📌 Example Code:

#include <iostream>
using namespace std;

class Person {
public:
    string name;
    int age;

    // Constructor
    Person(string n, int a) {
        name = n;
        age = a;
        cout << "Constructor called for " << name << endl;
    }

    void display() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
};

int main() {
    Person p1("Alice", 25);
    p1.display();

    return 0;
}

🔹 What is a Destructor?

  • A destructor is a special function that is automatically called when an object is destroyed.
  • It has the same name as the class but is preceded by a tilde (~).

📌 Example Code:

#include <iostream>
using namespace std;

class Person {
public:
    string name;

    // Constructor
    Person(string n) {
        name = n;
        cout << "Constructor called for " << name << endl;
    }

    // Destructor
    ~Person() {
        cout << "Destructor called for " << name << endl;
    }
};

int main() {
    Person p1("Bob");
    return 0;
}

🔹 Types of Constructors

1️⃣ Default Constructor

A constructor with no parameters.

#include <iostream>
using namespace std;

class Example {
public:
    Example() { // Default Constructor
        cout << "Default Constructor called!" << endl;
    }
};

int main() {
    Example obj; // Constructor is called automatically
    return 0;
}

2️⃣ Parameterized Constructor

A constructor that takes arguments.

#include <iostream>
using namespace std;

class Car {
public:
    string brand;

    // Parameterized Constructor
    Car(string b) {
        brand = b;
        cout << "Car brand: " << brand << endl;
    }
};

int main() {
    Car c1("Toyota");
    Car c2("Honda");
    return 0;
}

3️⃣ Copy Constructor

A constructor that copies values from one object to another.

#include <iostream>
using namespace std;

class Number {
public:
    int value;

    // Parameterized Constructor
    Number(int v) { 
        value = v;
    }

    // Copy Constructor
    Number(const Number &obj) {
        value = obj.value;
    }

    void display() {
        cout << "Value: " << value << endl;
    }
};

int main() {
    Number num1(10);
    Number num2 = num1; // Calls copy constructor

    num2.display();
    return 0;
}

✅ Why Use Constructors and Destructors?

  • Automatic Initialization → Constructors initialize objects automatically.
  • Memory Management → Destructors free memory when objects are destroyed.
  • Code Simplicity → Avoids manual setup and cleanup of objects.

🚀 Demonstrating Constructor and Destructor Execution

Let's see how constructors and destructors work together.

📌 Example Code:

#include <iostream>
using namespace std;

class Demo {
public:
    // Constructor
    Demo() {
        cout << "Constructor executed!" << endl;
    }

    // Destructor
    ~Demo() {
        cout << "Destructor executed!" << endl;
    }
};

int main() {
    cout << "Creating an object..." << endl;
    Demo obj;  // Constructor is called
    cout << "Object is about to be destroyed..." << endl;
    
    return 0; // Destructor is called automatically
}

Understanding constructors and destructors in C++ helps in efficient memory management and simplifies object initialization!

2️⃣ Loops in C++

Loops are used to repeat a block of code multiple times. C++ has three main loops:

🔹 For Loop

A for loop is used when we know how many times we want to repeat something.

#include <iostream>
              using namespace std;
              
              int main() {
                  for (int i = 0; i < 5; i++) {
                      cout << i << endl;
                  }
                  return 0;
              }

How it works?

  • int i = 0; initializes the loop variable.
  • i < 5; is the condition that keeps the loop running.
  • i++; increases i by 1 in each iteration.
  • The loop runs 5 times, printing numbers 0 to 4 one by one.

💡 Use case: Use a for loop when you want to repeat a task a set number of times, like printing numbers or iterating over an array.

🔹 While Loop

A while loop runs as long as a condition is true.

#include <iostream>
              using namespace std;
              
              int main() {
                  int x = 0;
                  while (x < 5) {
                      cout << x << endl;
                      x++;
                  }
                  return 0;
              }

How it works?

  • The loop starts with x = 0.
  • The condition x < 5 is checked before each loop run.
  • Inside the loop, x is printed, and then increased by 1 x++;.
  • The loop stops when x becomes 5.

💡 Use case: Use a while loop when you don't know how many times the loop should run, like waiting for user input.

Functions in C++

A function in C++ is a reusable block of code that performs a specific task. Instead of writing the same code again and again, we can use functions to make our programs shorter and easier to understand.

📌 Example Code:

#include <iostream>
                          using namespace std;
                          
                          // Function definition
                          string greet(string name) {
                              return "Hello, " + name;
                          }
                          
                          int main() {
                              cout << greet("Shifa") << endl;
                              return 0;
                          }

🔹 How This Code Works?

  • The function greet(string name) is defined before main().
  • It takes one input parameter (name).
  • Inside the function, it returns "Hello, " + name, which means it adds the name to "Hello, ".
  • We call the function using cout << greet("Shifa").
  • This prints "Hello, Shifa" on the screen.

✅ Why Use Functions?

  • ⭐ Reusability: We can call the function many times instead of writing the same code again.
  • ⭐ Better Readability: Code looks cleaner and is easier to understand.
  • ⭐ Easier to Fix Errors: If something goes wrong, we only need to change the function instead of the entire program.

🚀 Example: Calling the Function Multiple Times

#include <iostream>
                          using namespace std;
                          
                          string greet(string name) {
                              return "Hello, " + name;
                          }
                          
                          int main() {
                              cout << greet("Alice") << endl;
                              cout << greet("Bob") << endl;
                              cout << greet("Charlie") << endl;
                              return 0;
                          }

Output:

Hello, Alice
                          Hello, Bob
                          Hello, Charlie

This shows how functions help us avoid repeating code! 🎯

Arrays and Pointers in C++

In C++, arrays and pointers are closely related. Arrays store multiple elements in a contiguous memory block, while pointers allow direct memory access and manipulation.

🔹 What are Arrays?

  • Array → A collection of elements of the same type stored in memory.
  • Arrays use zero-based indexing, meaning the first element is at index 0.

📌 Example Code:

#include <iostream>
                          using namespace std;
                          
                          int main() {
                              int numbers[5] = {10, 20, 30, 40, 50};
                          
                              cout << "First element: " << numbers[0] << endl;
                              cout << "Second element: " << numbers[1] << endl;
                          
                              return 0;
                          }

🔹 What are Pointers?

  • Pointer → A variable that stores the memory address of another variable.
  • Pointers allow efficient memory manipulation and dynamic allocation.

📌 Example Code:

#include <iostream>
                          using namespace std;
                          
                          int main() {
                              int num = 10;
                              int* ptr = # // Pointer stores address of num
                          
                              cout << "Value of num: " << num << endl;
                              cout << "Address of num: " << &num << endl;
                              cout << "Pointer holds address: " << ptr << endl;
                              cout << "Value at pointer: " << *ptr << endl; // Dereferencing
                          
                              return 0;
                          }

🔹 Relationship Between Arrays and Pointers

  • In C++, an array name acts as a pointer to the first element.
  • Using pointer arithmetic, we can traverse the array efficiently.

📌 Example Code:

#include <iostream>
                          using namespace std;
                          
                          int main() {
                              int arr[3] = {10, 20, 30};
                              int* ptr = arr; // Points to first element
                          
                              cout << "First element: " << *ptr << endl;
                              cout << "Second element: " << *(ptr + 1) << endl;
                          
                              return 0;
                          }

✅ Benefits of Using Pointers with Arrays

  • Efficiency → Direct memory access speeds up operations.
  • Flexibility → Enables dynamic memory allocation.
  • Optimized Code → Reduces redundancy in accessing elements.

🚀 Dynamic Memory Allocation

We can use pointers to create arrays dynamically at runtime using new.

📌 Example Code:

#include <iostream>
                          using namespace std;
                          
                          int main() {
                              int* arr = new int[3]; // Dynamic array allocation
                          
                              arr[0] = 10;
                              arr[1] = 20;
                              arr[2] = 30;
                          
                              cout << "Dynamic Array: " << arr[0] << ", " << arr[1] << ", " << arr[2] << endl;
                          
                              delete[] arr; // Free allocated memory
                          
                              return 0;
                          }

Understanding arrays and pointers in C++ helps in optimizing memory usage and writing efficient programs!

Structures and Classes in C++

In C++, both structures (struct) and classes (class) are used to group related variables and functions. The main difference is that struct members are public by default, while class members are private by default.

🔹 What is a Structure?

  • A structure (struct) is a user-defined data type that groups different variables.
  • By default, all members of a struct are public.

📌 Example Code:

#include <iostream>
using namespace std;

struct Person {
    string name;
    int age;
};

int main() {
    Person p1 = {"Alice", 25};

    cout << "Name: " << p1.name << endl;
    cout << "Age: " << p1.age << endl;

    return 0;
}

🔹 What is a Class?

  • A class is an advanced version of a structure with private members by default.
  • It can have both data members (variables) and member functions (methods).

📌 Example Code:

#include <iostream>
using namespace std;

class Person {
private:
    string name;
    int age;

public:
    void setData(string n, int a) {
        name = n;
        age = a;
    }

    void display() {
        cout << "Name: " << name << endl;
        cout << "Age: " << age << endl;
    }
};

int main() {
    Person p1;
    p1.setData("Bob", 30);
    p1.display();

    return 0;
}

🔹 Differences Between Structures and Classes

Feature Structure (struct) Class (class)
Default Access Modifier Public Private
Encapsulation Limited Full (Data Hiding)
Usage Simple Data Grouping Object-Oriented Programming (OOP)

✅ Why Use Classes?

  • Encapsulation → Keeps data secure by using private members.
  • Reusability → We can create multiple objects from a single class.
  • Scalability → Classes support inheritance and polymorphism.

🚀 Creating Multiple Objects

We can create multiple objects using a class, just like with a structure.

📌 Example Code:

#include <iostream>
using namespace std;

class Person {
private:
    string name;
    int age;

public:
    void setData(string n, int a) {
        name = n;
        age = a;
    }

    void display() {
        cout << "Name: " << name << endl;
        cout << "Age: " << age << endl;
    }
};

int main() {
    Person p1, p2;
    p1.setData("Alice", 22);
    p2.setData("Charlie", 28);

    p1.display();  // Output: Name: Alice, Age: 22
    p2.display();  // Output: Name: Charlie, Age: 28

    return 0;
}

Both structures and classes are powerful tools in C++. Structures are useful for simple data grouping, while classes provide full object-oriented capabilities.

Object-Oriented Programming (OOP) in C++

Object-Oriented Programming (OOP) is a way to organize code by creating objects. An object is like a real-world item, such as a car or a person. These objects have attributes (characteristics) and methods (actions).

🔹 What are Classes and Objects?

  • Class → A template used to create objects.
  • Object → A specific example created from a class.

📌 Example Code:

#include <iostream>
                                          using namespace std;
                                          
                                          class Person {
                                          public:
                                              string name;
                                              int age;
                                          
                                              // Constructor
                                              Person(string n, int a) {
                                                  name = n;
                                                  age = a;
                                              }
                                          
                                              // Method
                                              void greet() {
                                                  cout << "Hello, my name is " << name << endl;
                                              }
                                          };
                                          
                                          int main() {
                                              Person p("Shifa", 20);
                                              p.greet();
                                              return 0;
                                          }

🔹 Explanation:

  • class Person creates a blueprint for a person.
  • The constructor Person(string n, int a) initializes the object’s attributes.
  • name and age store the person’s name and age.
  • The greet() method prints a message with the person’s name.
  • Person p("Shifa", 20) creates a person named Shifa, who is 20 years old.
  • p.greet() calls the function and prints "Hello, my name is Shifa".

✅ Why Use OOP?

  • Easy to Understand → Code is organized into objects, like real-world things.
  • Reusable → You can create multiple objects from the same class.
  • Less Repetition → Functions inside classes make code shorter.

🚀 Creating Multiple Objects
