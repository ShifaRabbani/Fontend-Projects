CSS Basics for Beginners

Welcome to AICode Lab's CSS tutorial for beginners! Cascading Style Sheets (CSS) is what makes the web beautiful. While HTML provides the structure of a webpage, CSS is responsible for how it looks.

In this guide, we'll cover the fundamentals of CSS with simple examples that you can experiment with right away.

What is CSS?

CSS stands for Cascading Style Sheets. It's a language used to describe the presentation of HTML documents. With CSS, you can control the layout, colors, fonts, and much more.

How to Add CSS to HTML

There are three ways to add CSS to an HTML document:

1. Inline CSS

Using the style attribute directly in HTML elements:

<p style="color: blue; font-size: 18px;">This is a blue paragraph with 18px font size.</p>

This is a blue paragraph with 18px font size.

2. Internal CSS

Using the <style> element in the <head> section of an HTML page:

<head>
  <style>
    p {
      color: green;
      font-size: 16px;
    }
  </style>
</head>
<body>
  <p>This paragraph would be green with 16px font size.</p>
</body>

3. External CSS

Creating a separate .css file and linking it using the <link> element:

<head>
  <link rel="stylesheet" href="styles.css">
</head>

This is the most recommended method for larger websites as it separates your HTML content from the CSS styling.

CSS Syntax

CSS syntax consists of a selector and a declaration block:

selector {
  property: value;
  another-property: another-value;
}
  • Selector: Points to the HTML element you want to style
  • Declaration Block: Contains one or more declarations separated by semicolons
  • Property: The style attribute you want to change
  • Value: The value for the property

CSS Selectors

Selectors are patterns used to select and style HTML elements. Here are the most common ones:

Element Selector

Selects all elements with the specified tag name:

p {
  color: red;
}

This paragraph would be red.

Class Selector

Selects all elements with the specified class attribute:

.highlight {
  background-color: yellow;
  font-weight: bold;
}

This paragraph has the highlight class.

ID Selector

Selects the element with the specified id attribute:

#header {
  background-color: black;
  color: white;
}

CSS Colors

CSS offers several ways to specify colors:

Color Names

p {
  color: red;
}

This text is red.

RGB Values

p {
  color: rgb(0, 0, 255);
}

This text is blue using RGB.

HEX Values

p {
  color: #00FF00;
}

This text is green using HEX.

Text Styling

CSS provides properties to style text:

p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  text-align: center;
  text-decoration: underline;
  line-height: 1.5;
}

This text has various styling properties applied.

CSS Box Model

The CSS box model is a fundamental concept that describes how elements are sized, spaced, and positioned. It consists of:

  • Content: The actual content of the element
  • Padding: Space between the content and the border
  • Border: A border around the padding
  • Margin: Space outside the border

Border Example

div {
  border: 2px solid black;
  padding: 10px;
  margin: 10px 0;
}
This div has a border, padding, and margin.

Margin Example

div {
  margin: 20px;
  border: 1px solid blue;
  background-color: #e6f2ff;
}
This div has a margin of 20px.

Padding Example

div {
  padding: 20px;
  border: 1px solid green;
  background-color: #e6ffe6;
}
This div has padding of 20px.

Responsive Design

Responsive design ensures that your website looks good on all devices. Media queries are the foundation of responsive design:

/* For mobile phones */
@media screen and (max-width: 768px) {
  body {
    font-size: 14px;
  }
  
  .container {
    width: 100%;
  }
}

CSS Transitions

Transitions allow elements to change values smoothly:

button {
  background-color: blue;
  color: white;
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: darkblue;
}

Common CSS Properties

Here's a table of commonly used CSS properties:

Property Description Example
color Sets text color color: blue;
background-color Sets background color background-color: #f0f0f0;
font-family Sets font type font-family: Arial, sans-serif;
font-size Sets font size font-size: 16px;
margin Sets margin area margin: 10px 5px;
padding Sets padding area padding: 10px;
border Sets border border: 1px solid black;
width Sets width width: 100px;
height Sets height height: 200px;
display Sets display type display: flex;
